Terms defination

  1. Javascript (JS)

    JavaScript is the programming language of the Web.

  2. Operators

    JavaScript operators are symbols that are used to perform operations on operands.

  3. Variables

    A JavaScript variable is simply a name of storage location. There are two types of variables in JavaScript : local variable and global variable.

  4. Variable naming conventions

      Name must start with a letter (a to z or A to Z), underscore( _ ), or dollar( $ ) sign.

      After first letter we can use digits (0 to 9), for example value1.

      JavaScript variables are case sensitive, for example x and X are different variables.

  5. Function

    A JavaScript function is a block of code designed to perform a particular task.

  6. String

    The JavaScript string is an object that represents a sequence of characters.

  7. Methods

    A JavaScript method is a property containing a function definition. Methods are functions stored as object properties.

  8. Arguments

    arguments is an Array -like object accessible inside functions that contains the values of the arguments passed to that function.

  9. Parameters

    parameters are the names listed in the function definition.

  10. Return

    The return statement stops the execution of a function and returns a value from that function

  11. Chaining methods

    Chaining Methods, also known as Cascading, means repeatedly calling one method after another on an object, in one continuous line of code. Let us look at an example where method chaining can help us avoid repetition.

  12. Booleans

    JavaScript booleans can have one of two values: true or false.

  13. Undefined

    undefined is a primitive value automatically assigned to variables that have just been declared, or to formal arguments for which there are no actual arguments.

  14. NaN

    The NaN property represents "Not-a-Number" value. This property indicates that a value is not a legal number.

  15. Escape

    This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

  16. Alert

    The alert() method displays an alert box with a specified message and an OK button.

  17. Comments

    JavaScript comments can be used to explain JavaScript code, and to make it more readable.

  18. jQuery

    jQuery is a JavaScript Library.

  19. Attributes

    Attributes are defined by HTML. Properties are accessed from DOM (Document Object Model) nodes.
